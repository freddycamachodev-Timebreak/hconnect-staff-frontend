# HCONNECT Staff Frontend

Administrative and staff portal for the HCONNECT platform.

This application allows hotel staff to:

* Monitor guest conversations
* Respond in real-time
* Receive escalated issues
* Coordinate support workflows
* Manage communication channels

---

## Technologies

* Next.js
* React
* TypeScript
* WebSockets / Socket.IO
* AWS integrations
* Real-time communication

---

## Installation

```bash
npm install
```

---

## Run Development Server

```bash
npm run dev
```

Default local environment:

```bash
http://localhost:3001
```

---

## Features

* Real-time guest monitoring
* Typing indicators
* Staff communication tools
* AI-assisted workflows
* Conversation visibility
* Multi-session support
* Scalable architecture

---

## Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_API_URL=http://localhost:3000
```
